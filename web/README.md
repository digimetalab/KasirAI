# KasirAI - Smart POS System

KasirAI is a modern, responsive implementation of a Point of Sale (POS) system designed for efficiency and aesthetics. It features role-based access for Cashiers, Owners, and Admins.

## Tampilan Antarmuka (Interface)

Berikut adalah tampilan dari 3 menu utama aplikasi: **POS (Kasir)**, **Dashboard (Owner)**, dan **Admin Panel**.
